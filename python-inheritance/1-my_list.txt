Créer une instance de MyList et ajouter des éléments :
>>> my_list = MyList()
>>> my_list.append(5)
>>> my_list.append(1)
>>> my_list.append(3)
>>> my_list.append(2)
>>> my_list.append(4)

Vérifier le contenu initial de la liste :
>>> print(my_list)
[5, 1, 3, 2, 4]

Tester la méthode print_sorted :
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]

Vérifier que la liste originale n'est pas modifiée :
>>> print(my_list)
[5, 1, 3, 2, 4]
